# Images Directory

This directory contains screenshots and assets for the SkinAura project.

## Required Images

Please add the following images to this directory:

1. **logo.png** - The SkinAura logo (metallic rose gold/copper colored logo with face icon)
2. **homepage.png** - Screenshot of the homepage interface with navigation and hero section
3. **upload.png** - Screenshot showing the image upload interface
4. **results.png** - Screenshot displaying disease prediction results
5. **chatbot.png** - Screenshot of the AI chatbot interaction

## Image Guidelines

- **Format**: PNG or JPG
- **Recommended Size**: 
  - Logo: 200x200px or higher
  - Screenshots: 1200x800px or higher
- **Naming**: Use lowercase with underscores (e.g., `homepage.png`)

## Note

These images are referenced in the main README.md file. Make sure the filenames match exactly as shown above.
