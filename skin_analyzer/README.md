# SkinAura Application

This directory contains the main application code for SkinAura.

## Files

- `app.py` - Main Streamlit application
- `hf.py` - HuggingFace integration (optional)
- `dat.json` - Disease information database
- `requirements.txt` - Python dependencies

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Create `.env` file with your API keys
3. Add model file: `final_vgg1920epochs.h5`
4. Run: `streamlit run app.py`

For detailed setup instructions, see the [main README](../README.md).
